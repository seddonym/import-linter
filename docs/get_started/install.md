# Install

Install `import-linter` using your favorite Python package manager. (You probably only want it as a dev dependency.)

=== "pip"
    ```console
    pip install import-linter
    ```

=== "uv"
    ```console
    uv add --dev import-linter
    ```

=== "poetry"
    ```console
    poetry add import-linter --group dev
    ```
